  • Robot simulator

  • 2D simulator of a wheeled robot coded in Python, using the library pygame. This project was done in a group of 3 people. We implemented an evolutionary algorithm that runs each robot in several rooms at each generation to use the average fitness as a technique to reach convergence faster. It also uses karman filters for self-localazition.

  • Auctioneer

    Auction analyzer

    Different types of auction mechanisms exist, and each of them has certain advantages and disadvantages. An “ideal” auction mechanism should maximize profits of sellers and buyers, maintain price stability, be “fair” and non-biased.

    This program simulates a modified second-price sealed-bid (Vickrey) auction scenario, both "pure" and level commitment version. The purpose of it is studying how "ideal" this kind of auctions and some bidding techniques are. To do so, it generates as output the statistics of market price, the sellers profit and the buyers profit. This was a project done in a group of 4 people.

  • TVA

    Tactical Voting Analyst

    Strategic voting is common in political elections and there are various famous examples for this from different countries (see in References section). Moreover, because of the increasingly common usage of electronic voting and the risk of influencing the outcome through illegally manipulating the voters’ preferences, the topic of strategic voting is receiving steadily growing attention since some years.

    This program simulates an election and analyses the possible ways of manipulating it considering the voters preferences, the kind of elections and if bullet voting is allowed. This was a project done in a group of 4 people.
