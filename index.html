  • Gesture-driven presentations

    No lecturer or student likes it when they hold a presentation and they need to run back and forth in the room just to press that keyboard button only to change a slide. And, surely, noone likes it when they forget their presentation pointer at home or their pointer runs out of battery while defending their thesis.

    The purpose of this research project is developing a system that can track the presenter and read his posture and gestures to make changes in the presentation. The main constraint is that the information can only be collected through the webcam of the laptop of the presenter. This also creates a power constrain.

    This is an ongoing project in a private repository

  • Robot simulator

    2D simulator of a wheeled robot coded in Python, using the library pygame. This project was done in a group of 3 people. We implemented an evolutionary algorithm that runs each robot in several rooms at each generation to use the average fitness as a technique to reach convergence faster. It also uses karman filters for self-localazition.

  • Predicting FOREX Market trends

    Making money out of money

    Trading in Forex can lead to profit if you react well (by buying and selling money) to dynamic changes in the exchange rates. However, forecasting of the Forex market is very difficult, due to its size, dispersion, and continuous operation.

    Therefore, the goal of this research project was to investigate whether neural networks and deep learning could be used to achieve an accurate data-driven forecast of the Forex market. This was a project done in a 6 people group.

  • ABO

    Alpha-Beta framework for Omega

    The purpose of this project was designing the board game Omega and creating an alpha-beta framework to play it. The main techniques used to improve the search are: Windows, Transposition tables, Aspiration search, Iterative deepening, Union Find, Move ordering and Killer move.

  • Auctioneer

    Auction analyzer

    Different types of auction mechanisms exist, and each of them has certain advantages and disadvantages. An “ideal” auction mechanism should maximize profits of sellers and buyers, maintain price stability, be “fair” and non-biased.

    This program simulates a modified second-price sealed-bid (Vickrey) auction scenario, both "pure" and level commitment version. The purpose of it is studying how "ideal" this kind of auctions and some bidding techniques are. To do so, it generates as output the statistics of market price, the sellers profit and the buyers profit. This was a project done in a 4 people group.

  • TVA

    Tactical Voting Analyst

    Strategic voting is common in political elections and there are various famous examples for this from different countries (see in References section). Moreover, because of the increasingly common usage of electronic voting and the risk of influencing the outcome through illegally manipulating the voters’ preferences, the topic of strategic voting is receiving steadily growing attention since some years.

    This program simulates an election and analyses the possible ways of manipulating it considering the voters preferences, the kind of elections and if bullet voting is allowed. This was a project done in a 4 people group.
